# Develop

Install dependencies using `npm i`

Start application:
```bash
npm run dev

# Alternative (using the deployed API instead of a local one)
npm run start
```

# Deploy

```bash
# Dev
npm run deploy-dev

# Prod
npm run deploy
```

# Show logs of deployed app

```bash
 # Dev
npm run logs-dev

 # Prod
npm run logs
```
